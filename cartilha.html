02

O que é Programar?

Programar

É ensinar o computador a executar tarefas usando instruções (código).

Lógica de Programação

É o raciocínio que organiza essas instruções.

Algoritmo

É o passo a passo que descreve como resolver um problema.

Linguagem de Programação

É o idioma usado para escrever o algoritmo — cada linguagem tem sua forma de se comunicar com o computador.

11

Glossário do Programador

Conceitos Fundamentais

Algoritmo Sequência lógica de passos para resolver um problema.
Variável Espaço de memória usado para armazenar valores.
Constante Valor que não muda durante o programa.
Inteiro (int) Número sem decimais.
Float / Double Número com casas decimais.
String Texto (ex: "Go Up").
Booleano (bool) Verdadeiro ou falso.
Função Conjunto de instruções reutilizável.
Loop Repete um bloco de código (for, while).
Array / Lista Grupo de valores em sequência.
Bug Erro no código.
Debug Localizar e corrigir erros.

Conceitos Web e APIs

HTTP / HTTPS Protocolos de comunicação da web.
API Interface de comunicação entre sistemas.
JSON Formato leve para troca de dados.
CRUD Criar, Ler, Atualizar e Excluir dados.
Token / JWT Código de autenticação seguro.
Deploy Colocar o sistema no ar.
12

Mentalidade de Programador

Errar é parte do processo.

O melhor código é o que resolve o problema.

Aprender a pesquisar é mais valioso que decorar comandos.

Cada erro corrigido é um novo aprendizado.

E lembre-se: subir de nível é sempre o objetivo.
